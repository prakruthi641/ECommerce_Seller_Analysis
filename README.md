# 📊 Ecommerce Seller Analysis Dashboard

## 📌 Project Overview

This project analyzes seller performance, revenue generation, and order fulfillment trends using the **Olist Brazilian E-commerce Dataset**. The analysis was first conducted in **Microsoft Excel** using Pivot Tables and KPI calculations, followed by the development of an interactive dashboard in **Power BI**.

The objective is to identify top-performing sellers, understand order delivery performance, and generate actionable business insights for marketplace optimization.

---

## 🎯 Business Objective

The primary goals of this analysis are:

- Evaluate seller performance across the marketplace.
- Identify top revenue-generating sellers.
- Analyze order delivery and cancellation trends.
- Understand seller contributions by city.
- Support data-driven business decisions.

---

## 📂 Dataset Information

Dataset: **Olist Brazilian E-commerce Dataset**

### Files Used

| File Name | Description |
|------------|------------|
| olist_orders_dataset.csv | Contains order information and order status |
| olist_order_items_dataset.csv | Contains product prices and seller details |
| olist_sellers_dataset.csv | Contains seller information and seller locations |

## 🔄 Project Workflow

### 1. Data Preparation in Excel

- Imported CSV datasets into Excel.
- Checked for missing values and duplicates.
- Performed data validation and cleaning.
- Created Pivot Tables for analysis.
- Calculated key performance metrics.

### 2. Dashboard Development in Power BI

- Imported cleaned data into Power BI.
- Created relationships between tables using:
  - `order_id`
  - `seller_id`
- Built DAX measures for KPIs.
- Designed an interactive dashboard for seller performance analysis.

## 💡 Key Insights

1. The vast majority of orders were successfully delivered, indicating strong fulfillment performance.

2. Order cancellation rates were very low, suggesting efficient order processing and customer satisfaction.

3. São Paulo sellers generated the highest seller activity and revenue contribution.

4. A few sellers showed comparatively higher cancellation rates and may require operational improvements.

5. Top-performing sellers contributed a significant share of overall sales and revenue.

---

## 🚀 Business Recommendations

- Reward and retain top-performing sellers through incentive programs.
- Monitor sellers with higher cancellation rates and identify operational bottlenecks.
- Support lower-performing sellers with targeted growth initiatives.
- Track delivery performance regularly to maintain customer satisfaction.
- Use seller performance metrics for strategic marketplace decision-making.
